# Skill · RAN Operations

An operations competency: the agent can read the state of a live radio access network, move a cell
site safely through its lifecycle, and optimize coverage and capacity — all expressed as well-formed,
risk-aware DOIL intents executed through MCP tools.

## Maps to certification
Required for [Network Operations — Silver](../certifications/network-operations-silver.md). Builds on
the foundational DOIL competency and exercises the same four-layer evaluation against the live-network
simulators.
