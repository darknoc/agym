# Skill · DOIL Authoring

A foundational competency: the agent can translate an operator's goal into well-formed DOIL intent,
execute it safely via MCP tools, and predict/critique its own outcome.

- [AGENG-101](../schools/agentic-engineering/departments/foundations/courses/ageng-101.md)

## Maps to certification
Required for [Agentic Foundations — Bronze](../certifications/agentic-foundations-bronze.md) and is a
prerequisite competency for all specialty schools. Mirrors the `doilCompliance` dimension of the
agun.ai `EvaluationScores`.
