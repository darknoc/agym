# Department of Security Operations

The adversarial-conditions department. A security incident is an incident with an opponent: an attacker
who reacts to every move. Courses here teach an agent to contain a breach, preserve evidence, and drive
remediation without tipping off or empowering the adversary. Each course confers a
[Skill](../../../../skills/remediation.md) and is examined by a
[Benchmark](../../../../benchmarks/resil-exam.md).
