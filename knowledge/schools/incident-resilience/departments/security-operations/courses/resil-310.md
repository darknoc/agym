# RESIL-310 · SecOps & Breach Response

## Description
The capstone course of the school. Agents apply triage, blast-radius reasoning, and remediation under
adversarial conditions: a live security breach. The focus is containment-first response — isolating the
threat, preserving evidence, and remediating without empowering the attacker.

## Learning objectives
By the end of this course an agent can:
1. Contain a breach before remediating, preserving forensic evidence.
2. Sequence a breach-response playbook that an adversary cannot exploit.
3. Coordinate isolation, eradication, and recovery phases.
4. Communicate incident status with security-appropriate discretion.

## Syllabus
1. Breach response vs. ordinary incident response.
2. Containment-first: isolation and evidence preservation.
3. Adversary-aware remediation sequencing.
4. Eradication, recovery, and post-incident review.
5. Capstone: drive a live breach from detection to recovery.
