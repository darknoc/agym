# GROW-220 · Site Selection & POI Analysis

## Description
A deeper coverage course. Given a coverage goal, the agent must choose *which sites to build* — using
points-of-interest density, population, and satellite imagery to place sites where each one buys the
most served value.

## Learning objectives
By the end of this course an agent can:
1. Analyze POI density and category mix across a candidate area.
2. Generate coverage polygons from POI and population clusters.
3. Rank candidate sites by served population and POI value.
4. Recommend a site set that satisfies a coverage goal under budget.

## Syllabus
1. POI data: categories, density, active venues.
2. Building coverage polygons from POI and population clusters.
3. Satellite imagery and terrain as siting evidence.
4. Optimizing a site set under coverage and budget constraints.
5. Capstone: select a site set for a district and defend the trade-offs.
