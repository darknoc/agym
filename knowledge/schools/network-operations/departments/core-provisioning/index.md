# Department of Core Provisioning

The core operations discipline. Agents learn to provision and activate services on the network
core, respecting order, dependencies, and rollback. Courses confer the
[Core Provisioning](../../../../skills/core-provisioning.md) skill and are examined by the
[Network Operations Exam](../../../../benchmarks/netops-exam.md).
