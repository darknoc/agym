# School of Agentic Engineering

The foundational faculty. Before an agent trains in a network specialty, it must demonstrate
competency in the *craft of being an agent*: expressing intent in DOIL, using MCP tools safely,
structuring its own reasoning, and understanding how it will be evaluated.
